# RClick 设置界面重构设计

> 最后更新：2026-04-06
> 状态：设计文档
> 架构：完全磁盘访问权限 + 瘦 Extension
> 风格：macOS 系统设置风格（分组卡片式布局）

---

## 一、设计目标

重新设计设置界面的"通用设置"部分，采用 macOS 系统设置风格（分组卡片式布局），并将"折叠上下文菜单"开关从通用设置移到每个分类里面。

**设计参考**: 用户提供的截图显示了 macOS 系统设置风格：
- 分组卡片式布局
- 每组有标题和说明文字
- 开关在右侧对齐
- 底部有危险操作（红色文字按钮）

---

## 二、通用设置新布局

通用设置采用三组卡片式布局：

### 2.1 第一组：主要控制

| 项目 | 类型 | 说明 |
|------|------|------|
| 启用「RClick」 | Toggle 开关 | 主开关，控制扩展启用/禁用 |
| 登录时启动 | Toggle 开关 | 系统登录时自动启动 RClick |

**注意**:
- "RClick"文案替换为"RClick"
- "登录时启动"移到"启用"之后

### 2.2 第二组：权限

| 项目 | 类型 | 说明 |
|------|------|------|
| Finder 扩展 | 状态显示 | 只读，带绿色勾选图标表示已启用 |
| 完全磁盘访问权限 | 按钮 | 点击跳转到系统设置请求权限 |
| 辅助功能 | 按钮 | 点击跳转到系统设置请求权限 |

### 2.3 第三组：设置管理

| 项目 | 类型 | 说明 |
|------|------|------|
| 备份 | 按钮组 | 导出设置 / 导入设置 |
| 日志 | 按钮 | 导出日志 |
| 重置所有设置 | 文字按钮 | 红色，底部右对齐，危险操作 |

## 三、各分类折叠开关

将"折叠上下文菜单"从通用设置移除，改为每个分类独立控制：

| 分类 | 开关名称 | 存储键 | 位置 |
|------|----------|--------|------|
| Apps | 折叠应用菜单 | `foldAppsMenu` | 列表顶部 |
| Actions | 折叠动作菜单 | `foldActionsMenu` | 列表顶部 |
| New File | 折叠新建文件菜单 | `foldNewFileMenu` | 列表顶部 |
| Common Dir | 折叠常用文件夹菜单 | `foldCommonDirMenu` | 列表顶部 |
| Scripts | 折叠脚本菜单 | `foldScriptsMenu` | 列表顶部 |

### 3.1 开关行为

- 默认状态：关闭（不折叠）
- 开启后：该分类的菜单项归入「RClick」子菜单
- 状态持久化：使用 `@AppStorage` 存储

## 五、设计规范

### 5.1 分组风格

- 使用 `Form` + `Section` 结构
- 每组有独立的标题和背景
- 组与组之间有明显分隔

### 5.2 图标状态

- 使用 SF Symbols 系统图标
- `checkmark.circle.fill` (绿色) - 表示已授权/已启用
- `circle` (灰色) - 表示未授权/未启用
- `exclamationmark.triangle` (黄色) - 表示警告/需要注意

### 5.3 开关位置

- 每个分类列表顶部或 header 区域
- 开关右侧对齐
- 开关旁有说明文字

### 5.4 数据存储

- 使用 `@AppStorage` 持久化存储折叠状态
- 键名统一使用驼峰命名

### 5.5 文案规范

- 所有文案使用中文
- "RClick"替换为"RClick"
- 说明文字使用 `.secondary` 颜色

## 八、脚本设置界面设计

### 9.1 界面布局

脚本设置界面采用列表 + 编辑器的布局：

| 区域 | 组件 | 说明 |
|------|------|------|
| 顶部 | 折叠开关 | "折叠脚本菜单"开关 |
| 列表区 | 脚本列表 | 显示所有脚本（名称、类型、启用状态） |
| 工具栏 | 添加/删除按钮 | 添加新脚本、删除选中脚本 |
| 编辑器 | 脚本编辑器 | 多行文本框，支持语法高亮（可选） |
| 底部 | 测试按钮 | 选中测试文件执行脚本 |

### 9.2 脚本列表

每行显示：
- 左侧：SF Symbol 图标（根据脚本类型）
- 中间：脚本名称 + 类型标签
- 右侧：启用/禁用开关
- 拖拽手柄：支持上下拖动排序

### 9.3 脚本编辑器

编辑字段：
- 名称（必填）
- 类型选择器（Shell / Python / AppleScript）
- 脚本内容（多行文本框）
- 图标选择器（SF Symbol）

### 9.4 预定义脚本模板

提供快速创建模板：
| 名称 | 类型 | 说明 |
|------|------|------|
| 显示文件信息 | shell | `mdls "$1"` |
| 在 VSCode 打开 | shell | `open -a "Visual Studio Code" "$@"` |
| 获取文件哈希 | shell | `shasum -a 256 "$@"` |

### 9.5 安全提示

在脚本设置界面顶部添加黄色警告横幅：
> ⚠️ 安全提示：脚本以当前用户权限运行，请确保脚本来源可靠。自行承担执行风险。
